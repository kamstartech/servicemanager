## Purpose

Quick reference documents are designed to be:
- **Concise** - 1-2 pages maximum
- **Focused** - One topic per document
- **Practical** - Common tasks and examples
- **Up-to-date** - Reflect current implementation

## Format

Quick references should include:
1. Brief overview (1-2 sentences)
2. Common tasks with examples
3. Key commands or code snippets
4. Links to detailed documentation
5. Troubleshooting tips (optional)
